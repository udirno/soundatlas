2. Create `frontend/src/lib/api.ts` with typed fetch helpers. Define TypeScript interfaces mirroring the backend Pydantic schemas:
   - `CountryListItem`: id, name, iso_alpha2, latitude (number|null), longitude (number|null), artist_count, track_count, top_genre (string|null)
   - `CountryDetail`: id, name, iso_alpha2, latitude, longitude, artists (ArtistListItem[]), genre_breakdown (Record<string,number>), audio_feature_averages (Record<string, number|null>)
   - `CountryComparison`: id, name, iso_alpha2, country_averages (Record<string, number|null>), global_averages (Record<string, number|null>)
   - `ArtistListItem`: id, name, spotify_id, country_id, genres (string[]|null), popularity (number|null), image_url (string|null)
   - `TrackListItem`: id, name, spotify_id, album_name (string|null), energy/danceability/valence/tempo/acousticness (number|null)

   Export async fetch functions that call the FastAPI backend (use relative URLs so Next.js rewrites or same-origin works in Docker):
   - `fetchCountries(): Promise<CountryListItem[]>` — GET `http://localhost:8000/api/countries`
   - `fetchCountryDetail(id: number): Promise<CountryDetail>` — GET `http://localhost:8000/api/countries/{id}`
   - `fetchCountryComparison(id: number): Promise<CountryComparison>` — GET `http://localhost:8000/api/countries/{id}/comparison`

   Use the base URL `process.env.NEXT_PUBLIC_API_URL ?? 'http://localhost:8000'` so it works both server-side and client-side. For server-side fetches (in page.tsx), use the full URL. For client-side fetches (in components), the same URL works since the browser can reach localhost:8000.

   Each function should: call fetch, check response.ok, throw on error, return typed JSON.

3. Create `frontend/src/lib/colors.ts` with a shared genre color palette:
   ```typescript
   export const GENRE_COLORS: Record<string, string> = {
     'pop': '#f43f5e',
     'hip hop': '#f97316',
     'rock': '#8b5cf6',
     'electronic': '#06b6d4',
     'r&b': '#10b981',
     'latin': '#eab308',
     'country': '#a3e635',
     'metal': '#dc2626',
     'jazz': '#d946ef',
     'classical': '#2dd4bf',
     'indie': '#fb923c',
     'soul': '#facc15',
   };

   export const FALLBACK_COLOR = '#94a3b8';

   export function getGenreColor(genre: string): string {
     const lower = genre.toLowerCase();
     for (const [key, color] of Object.entries(GENRE_COLORS)) {
       if (lower.includes(key)) return color;
     }
     return FALLBACK_COLOR;
   }
   ```

   The `getGenreColor` function uses `includes()` so that "hip hop" matches "southern hip hop", "pop" matches "k-pop", etc. This is important because Spotify genre tags are freeform and granular.

   Also export a function `buildMapboxColorExpression()` that returns the Mapbox `match` expression array for use in the circle layer paint property, built from GENRE_COLORS entries.

1. Create `frontend/src/components/MapView.tsx` as a `'use client'` component:
   - Import `mapboxgl` and `'mapbox-gl/dist/mapbox-gl.css'` at the top of the file.
   - Import `CountryListItem` from `@/lib/api` and `GENRE_COLORS, FALLBACK_COLOR` from `@/lib/colors`.
   - Props interface: `{ countries: CountryListItem[]; onCountrySelect?: (countryId: number) => void; }`
   - Use `useRef<HTMLDivElement>(null)` for the container and `useRef<mapboxgl.Map | null>(null)` for the map instance. NEVER put the map in useState.
   - In `useEffect` (empty deps array):
     a. Guard: if container is null or map already initialized, return.
     b. Set `mapboxgl.accessToken = process.env.NEXT_PUBLIC_MAPBOX_TOKEN!`
     c. Create map: `new mapboxgl.Map({ container, style: 'mapbox://styles/mapbox/dark-v11', center: [0, 20], zoom: 1.5, projection: 'mercator' })`
     d. On `map.on('load', () => { ... })`:
        - Build GeoJSON FeatureCollection from `countries` prop. Filter out entries where latitude or longitude is null. Each feature: geometry is Point with [longitude, latitude], properties include id, name, track_count, artist_count, top_genre (default to 'Unknown' if null).
        - `map.addSource('countries', { type: 'geojson', data: geojson })`
        - `map.addLayer({ id: 'country-circles', type: 'circle', source: 'countries', paint: { ... } })` with:
          - `circle-radius`: Use `interpolate` + `linear` on `sqrt` of `track_count`. Stops: sqrt(0)=0 -> 4px, sqrt(50)~7 -> 12px, sqrt(100)=10 -> 20px, sqrt(500)~22 -> 36px.
          - `circle-color`: Use `match` on `get('top_genre')` with entries from GENRE_COLORS. The match expression should be built from GENRE_COLORS keys. Fallback color: FALLBACK_COLOR.
          - `circle-opacity`: 0.85
          - `circle-stroke-width`: 1
          - `circle-stroke-color`: '#ffffff'
          - `circle-stroke-opacity`: 0.3
     e. Cleanup: return `() => { map.current?.remove(); map.current = null; }`
   - Render: `<div ref={mapContainer} className="w-full h-screen" />`
