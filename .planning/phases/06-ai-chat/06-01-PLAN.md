  <action>
    1. Add `anthropic>=0.80.0` to `backend/requirements.txt` (after the existing `redis==5.1.0` line).

    2. Create `backend/app/services/redis_client.py`:
       - Module-level `_redis: Optional[aioredis.Redis] = None` singleton
       - `async def get_redis() -> aioredis.Redis`: lazily creates connection via `aioredis.Redis.from_url(settings.REDIS_URL, decode_responses=True)`
       - `async def close_redis() -> None`: closes connection and sets `_redis = None`
       - Import `redis.asyncio as aioredis` (NOT the deprecated `aioredis` package)
       - Import settings from `app.config`

    3. Create `backend/app/services/ai_service.py`:
       - Module-level `_client: Optional[anthropic.AsyncAnthropic] = None` singleton
       - `def get_claude_client() -> anthropic.AsyncAnthropic`: lazily creates with `settings.ANTHROPIC_API_KEY`
       - Constants: `CACHE_TTL_SECONDS = 3600`, `MODEL = "claude-sonnet-4-5-20250929"`, `MAX_TOKENS = 1024`
       - `def _cache_key(question: str) -> str`: SHA-256 hash of `question.strip().lower()`, prefixed `"ai:ask:"`
       - `def _build_system_prompt() -> str`: Returns a system prompt instructing Claude it is a music library analyst for SoundAtlas. Tell it to answer questions based on the provided listening data context. Keep answers concise (2-3 paragraphs max). Mention specific numbers from the data when relevant.
       - `def _build_user_message(question: str, context: dict) -> str`: Format the context dict as a readable text block followed by the user's question. Include library_stats, top_countries, top_genres, and audio_feature_averages sections.
       - `async def ask(question: str, context: dict, db: AsyncSession) -> dict`:
         a. Get Redis via `redis_client.get_redis()`
         b. Check cache: `await redis.get(_cache_key(question))` — if hit, `json.loads()` it, set `cached=True`, return
         c. On miss: call `await client.messages.create(model=MODEL, max_tokens=MAX_TOKENS, system=system_prompt, messages=[{"role": "user", "content": user_content}])`
         d. Extract `response.content[0].text` as answer, `response.usage.input_tokens + response.usage.output_tokens` as total_tokens
         e. Measure elapsed time with `time.perf_counter()` before/after the Claude call
         f. Cache: `await redis.setex(key, CACHE_TTL_SECONDS, json.dumps({"answer": answer, "cached": False, "tokens": total_tokens}))`
         g. Log: `await db.execute(insert(AIQueryLog).values(query=question, response=answer, model_name=MODEL, token_count=total_tokens, response_time_ms=elapsed_ms))` then `await db.commit()`
         h. Return `{"answer": answer, "cached": False, "tokens": total_tokens}`

    Use `AsyncAnthropic` (NOT sync `Anthropic`) — the backend is fully async.
    Only log to ai_query_log on cache MISS (cache hits made no Claude call, so no tokens/timing to log).
  </action>

  </verify>
