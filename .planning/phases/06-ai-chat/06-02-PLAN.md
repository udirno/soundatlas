  <action>
    1. Add to `frontend/src/lib/api.ts` (append to end of file):
       - `AIAskResponse` interface: `{ answer: string; query: string; sources: string[]; cached?: boolean; tokens?: number }`
       - `AISuggestion` interface: `{ question: string }`
       - `async function askAI(question: string): Promise<AIAskResponse>` — POST to `${getBaseUrl()}/api/ai/ask` with JSON body `{ question }`, throw on non-ok
       - `async function fetchAISuggestions(): Promise<AISuggestion[]>` — GET `${getBaseUrl()}/api/ai/suggestions` with `cache: 'no-store'`, throw on non-ok
       - Export both functions and interfaces

    2. Create `frontend/src/components/AIChatPanel.tsx`:
       - 'use client' directive
       - Props: `{ onClose: () => void }`
       - State: `messages: Array<{role: 'user'|'assistant', content: string}>`, `suggestions: AISuggestion[]`, `input: string`, `isLoading: boolean`
       - `useEffect` on mount: fetch suggestions via `fetchAISuggestions().then(setSuggestions).catch(() => {})`
       - `useRef` for messages container (auto-scroll to bottom on new message)
       - `handleSend(question?: string)` function:
         a. Get text from `question ?? input`, trim, return if empty or isLoading
         b. Clear input, append user message to messages
         c. Set isLoading=true
         d. Call `askAI(text)`, on success append assistant message, on error append error message "Sorry, I couldn't generate an answer. Please try again."
         e. Set isLoading=false in finally block
       - Layout: `fixed bottom-4 right-4 w-96 h-[580px] z-50` — floating panel, bottom-right, avoids CountryPanel (top-right) and StatsSidebar (top-left)
       - Structure:
         - Header bar: "Ask AI" title + close button (X icon from lucide-react)
         - Messages area (flex-1, overflow-y-auto, scrollable):
           - If `messages.length === 0` AND suggestions loaded: render suggestion chips as clickable rounded buttons (e.g., `bg-blue-500/20 text-blue-300 rounded-full px-3 py-1.5 text-sm cursor-pointer hover:bg-blue-500/30`). On click, call `handleSend(chip.question)`.
           - Otherwise: render message bubbles. User messages right-aligned (bg-blue-600), assistant messages left-aligned (bg-gray-700). Use `whitespace-pre-wrap` on assistant messages to preserve Claude's formatting (newlines, paragraphs).
           - If isLoading: show a pulsing "Thinking..." indicator after the last message
         - Input area at bottom: text input + send button. Submit on Enter key or click send. Disable input and button while isLoading.
       - Color scheme: dark theme matching the app (bg-gray-900, text-white, border-gray-700)
       - Add `rounded-xl shadow-2xl border border-gray-700 overflow-hidden` for the panel container
       - Auto-scroll: `useEffect` that scrolls messagesEndRef into view when messages array changes
  </action>
