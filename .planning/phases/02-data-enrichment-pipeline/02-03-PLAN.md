  <action>
Create `pipeline/run_pipeline.py` that:

1. **Orchestrates the full pipeline** by running steps in order:
   - Step 1: `seed_countries.py` (idempotent — safe to re-run)
   - Step 2: `seed_library.py` (idempotent — seeds artists + tracks)
   - Step 3: `enrich_spotify.py` (idempotent — resolves spotify_id + metadata)
   - Step 4: `enrich_musicbrainz.py` (idempotent — resolves countries)

   Run each step via `subprocess.run([sys.executable, script_path, "--env-file", env_file], check=True)`. If any step fails (non-zero exit), log the error and stop (do not proceed to next step). Each step is independently resumable, so re-running the orchestrator after a failure picks up where it left off.

2. **Stats logging after completion** — connect to PostgreSQL and query:
   ```sql
   SELECT COUNT(*) FROM artists;
   SELECT COUNT(*) FROM tracks;
   SELECT COUNT(*) FROM user_tracks;
   SELECT COUNT(*) FROM artists WHERE spotify_id IS NOT NULL;
   SELECT COUNT(*) FROM artists WHERE genres IS NOT NULL;
   SELECT COUNT(*) FROM artists WHERE image_url IS NOT NULL;
   SELECT mb_resolution_status, COUNT(*) FROM artists GROUP BY mb_resolution_status;
   SELECT COUNT(*) FROM artists WHERE country_id IS NOT NULL;
   SELECT COUNT(DISTINCT country_id) FROM artists WHERE country_id IS NOT NULL;
   ```

3. **Print formatted stats table:**
   ```
   ==================================================
   SoundAtlas Pipeline Stats
   ==================================================
     Total artists:         3,022
     Total tracks:          9,115
     Total user_tracks:     9,115

     Spotify ID resolved:   2,890 / 3,022  (95.6%)
     Genres populated:      2,890 / 3,022  (95.6%)
     Images populated:      2,750 / 3,022  (91.0%)

     MusicBrainz Results:
       resolved:            1,850  (61.3%)
       not_found:           1,172  (38.7%)
       pending:                 0  (0.0%)

     Countries represented: 45

     Duration:              52m 30s
   ==================================================
   ```
   (Numbers above are illustrative — actual values from DB queries.)

4. **Track total duration** using `time.time()` at start and end.

5. **CLI entrypoint** with:
   - `--env-file` arg (passed through to all sub-scripts)
   - `--export-path` arg (passed to seed_library.py)
   - `--stats-only` flag: if set, skip all pipeline steps and just print stats from current DB state
   - `--skip-musicbrainz` flag: if set, skip step 4 (useful for quick testing without the ~50min MB run)

6. Pass `--export-path` only to `seed_library.py`. Other scripts don't need it.

IMPORTANT: All subprocess calls must pass through the env-file argument. Each sub-script handles its own database connection.
