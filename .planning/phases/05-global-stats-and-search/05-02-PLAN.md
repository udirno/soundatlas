  2. Modify `frontend/src/components/MapView.tsx`:
     - Add to `MapViewProps`: `flyToTarget?: { lng: number; lat: number } | null` and `onFlyToComplete?: () => void`
     - Add a NEW `useEffect` (separate from the map init useEffect) that watches `flyToTarget`:
       ```typescript
       useEffect(() => {
         if (!flyToTarget || !map.current) return;
         map.current.flyTo({
           center: [flyToTarget.lng, flyToTarget.lat],
           zoom: Math.max(map.current.getZoom(), 4),
           duration: 1200,
         });
         onFlyToComplete?.();
       }, [flyToTarget, onFlyToComplete]);
       ```
     - This avoids forwardRef + dynamic import complexity (per research recommendation). The prop-based pattern is simpler and works with `next/dynamic`.
     - Keep the existing click-to-fly behavior unchanged.
