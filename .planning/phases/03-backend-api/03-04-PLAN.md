**`calculate_diversity_score(country_artist_counts: list[int]) -> float`** (pure function, not async):
- Shannon entropy normalized to [0,1]
- `total = sum(counts)`, filter to non-zero counts
- If total == 0 or only 1 country, return 0.0
- `entropy = -sum((c/total) * math.log(c/total) for c in counts)`
- `max_entropy = math.log(n)` where n = number of countries with artists
- Return `round(entropy / max_entropy, 4)`
- Import `math` at module level

**`get_dashboard_stats(db: AsyncSession) -> dict`:**
- Query 1: Global counts
  ```python
  counts_stmt = select(
      func.count(distinct(Country.id)).label("country_count"),
      func.count(distinct(Artist.id)).label("artist_count"),
      func.count(distinct(Track.id)).label("track_count"),
  ).select_from(Country).outerjoin(Artist).outerjoin(Track)
  ```
- Query 2: Per-country artist counts for diversity score
  ```python
  per_country_stmt = (
      select(func.count(Artist.id).label("cnt"))
      .where(Artist.country_id.is_not(None))
      .group_by(Artist.country_id)
  )
  ```
  Call `calculate_diversity_score(list(per_country_scalars))`
- Query 3: Top genres (global) — load ALL artists with genres, flatten, Counter, take top 10
  ```python
  artists_stmt = select(Artist.genres).where(Artist.genres.is_not(None))
  rows = (await db.execute(artists_stmt)).scalars().all()
  all_genres = []
  for genres_list in rows:
      if genres_list:
          all_genres.extend(genres_list)
  top_genres = [{"genre": g, "count": c} for g, c in Counter(all_genres).most_common(10)]
  ```
- Query 4: Top countries by artist count — top 10
  ```python
  top_countries_stmt = (
      select(Country.id, Country.name, Country.iso_alpha2,
             func.count(Artist.id).label("artist_count"))
      .join(Artist, Artist.country_id == Country.id)
      .group_by(Country.id)
      .order_by(func.count(Artist.id).desc())
      .limit(10)
  )
  ```
- Return dict with: country_count, artist_count, track_count, diversity_score, top_genres, top_countries

**`get_genre_distribution(db: AsyncSession, country_id: int | None = None) -> dict`:**
- Global genres: use `text()` raw SQL for unnest:
  ```python
  global_stmt = text("""
      SELECT unnest(genres) AS genre, COUNT(*) AS count
      FROM artists
      WHERE genres IS NOT NULL
      GROUP BY genre
      ORDER BY count DESC
      LIMIT 20
  """)
  ```
- If country_id provided, same query with `WHERE country_id = :cid AND genres IS NOT NULL`
  Use `text(...).bindparams(cid=country_id)` for parameterized query.
- Return dict with: global_genres (list of {genre, count}), country_id, country_genres

**`get_feature_averages(db: AsyncSession, country_id: int | None = None) -> dict`:**
- Global averages: `SELECT avg(energy), avg(danceability), avg(valence), avg(tempo), avg(acousticness) FROM tracks`
- If country_id: same but JOIN artists WHERE artists.country_id = :cid
- Return dict with: global_averages, country_id, country_averages (or None if no country_id)

Use `from collections import Counter`, `import math`, `from sqlalchemy import select, func, distinct, text`.
